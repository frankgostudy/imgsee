# imgSee - 高對比專業圖片檢視器

imgSee 是一款基於 Python 和 PyQt6 開發的高性能圖片縮圖瀏覽與檢視工具。專為快速瀏覽大量圖片設計，並提供科技感的深色高對比介面。

## 🚀 主要特色
- **高對比科技風介面**：採用明亮青色 (#00d9ff) 與亮黃色 (#ffcc00) 配色，確保在深色模式下資訊清晰可見。
- **高性能縮圖預覽**：支援自定義縮圖尺寸（小、中、大），快速載入目錄圖片。
- **全螢幕沉浸模式**：支援一鍵進入全螢幕，專注於圖片細節。
- **專業交互體驗**：具備手形游標提示、滾輪精準縮放（10% 步進）及圖片拖曳平移。

## 🛠 快捷鍵說明

### 縮圖主介面
- **選擇資料夾**：點擊左側按鈕或直接在路徑框輸入。
- **調整大小**：右側下拉選單切換縮圖顯示尺寸。

### 大圖檢視模式
| 快捷鍵 | 功能說明 |
| :--- | :--- |
| **Enter / F11** | 切換/退出全螢幕模式 |
| **Esc** | 退出全螢幕 或 關閉大圖視窗 |
| **← / → (方向鍵)** | 上一張 / 下一張 圖片切換 |
| **滑鼠滾輪** | 每次 10% 比例縮放圖片 |
| **滑鼠左鍵拖曳** | 平移圖片（游標自動變更為抓取手形） |
